# Rock_Paper_Scissors
## 日本語
### 概要
シンプルなジャンケンゲーム
### ルール
+ 先に三勝した方の勝ち
### コンパイル方法
	gcc -Wextra -Werror -Wall Rock_Paper_Scissors.c
